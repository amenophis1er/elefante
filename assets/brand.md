# elefante brand

## Name

Always lowercase: **elefante** — never "Elefante" or "ELEFANTE".

## Colors

| Role       | Hex       | Usage                                    |
|------------|-----------|------------------------------------------|
| Primary    | `#7B7FBF` | Logo, active states, primary buttons     |
| Primary hover | `#6A6EAE` | Hover states on primary elements      |
| Primary light | `#7B7FBF` at 10% opacity | Active nav backgrounds, subtle highlights |
| Text dark  | `#2D2F45` | Wordmark, headings                       |
| Text       | `#44403C` | Body text (stone-700)                    |
| Text muted | `#78716C` | Secondary text (stone-500)               |
| Surface    | `#FAFAF9` | Page background (stone-50)               |
| Card       | `#FFFFFF` | Card backgrounds                         |
| Border     | `#E7E5E4` | Card borders (stone-200)                 |

## Logo

- **Mark**: Rounded square with geometric "e" letterform (`logo-mark.svg`)
- **Wordmark**: Mark + "elefante" text (`logo-wordmark.svg`)
- **Favicon**: Simplified mark for small sizes (`favicon.svg`)

The "e" in the mark is a circle with an open gap (evoking an elephant trunk without being literal) crossed by a horizontal bar. White on primary background.

## Typography

- UI: System font stack (`system-ui, -apple-system, sans-serif`)
- Code: `SF Mono, Fira Code, Fira Mono, Menlo, Consolas, monospace`
- Inline code: Pink pill style (`#C0392B` on `#FCE4E4` background)
