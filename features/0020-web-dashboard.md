Read-only web UI for browsing your vault. Like the GitHub file browser but purpose-built for memories.

Features:
- Browse memories by type, profile, tags
- Search with highlighting
- View memory content with rendered Markdown
- Timeline view (git log visualization)
- Memory stats (count by type, most accessed, staleness)

Stack: static site generated from the vault, or a local dev server (`elefante dashboard`).

Parked because: the vault is already browsable in GitHub UI, any editor, or the CLI. A dashboard is nice-to-have, not essential. Wait for user demand.
